# To-Do List em Java

Projeto simples de lista de tarefas desenvolvido em Java, com foco em
Programação Orientada a Objetos e boas práticas.

## Funcionalidades
- Adicionar tarefa
- Listar tarefas
- Marcar tarefa como concluída
- Editar tarefa
- Remover tarefa

## Tecnologias
- Java
- Scanner
- Collections (List, ArrayList)
- Stream API
- Optional

## Estrutura do Projeto
src/
├── Main.java
├── model/
│ └── Tarefa.java
└── service/
└── GerenciadorTarefas.java


## Como executar
1. Clone o repositório
2. Compile os arquivos Java
3. Execute a classe Main

javac src/Main.java
java Main

## Objetivo

Projeto criado para fins de aprendizado e prática de conceitos de Java e POO.
